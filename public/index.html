URL Shortener API

Coded by Autumn Bullard

My project for the URL Shortener Microservice challenge as part of the curriculum for the Apis and Microservices Certification on Free Code Camp.

View the source code for this project.

Project Usage

Submit a web address in the form below to get a shortened URL that will redirect to the original URL.

For example,

https://autumnchris-url-shortener.herokuapp.com/XepQ8lXk

will redirect to

https://github.com
